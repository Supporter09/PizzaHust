# PizzaHust

Web-only pizza ordering MVP. Course project.

## Where things live

- **Application/** — full app stack (frontend, backend, harness). Start here for any code work.
- **Documents/** — feasibility, planning, analysis (LaTeX), use case diagram (Mermaid), 8-week plan.
- **Week 1/** — ERD and data dictionary deliverables.

## Working on the app

Read `Application/AGENTS.md` first. It links to `PRODUCT.md`, `ARCHITECTURE.md`, `CONTRACTS.md`, and the verification workflow.

```sh
cd Application
./init.sh        # first-time bootstrap
./verify.sh      # gate before any feature is marked done
```
